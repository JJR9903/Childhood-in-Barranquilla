# Childhood in Barranquilla and Top 13 Colombian Cities

A comprehensive data analysis project exploring childhood conditions, educational outcomes, and early childhood development across Barranquilla and Colombia's top 13 cities.

This repository contains data cleaning and analysis workflows focused on understanding childhood experiences, educational quality, and early childhood development indicators in Barranquilla and selected Colombian cities. **Key Research Focus:**
